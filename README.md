# Микроконтроллерный сенсор с системой управления энергопотреблением

Проект автономного сенсорного узла, разработанного для работы в экстремальных условиях. Основная цель проекта — достижение максимальной автономности за счет использования литий-ионного конденсатора (LIC) и интеллектуальных алгоритмов управления питанием.

##  Описание проекта

Разработка направлена на решение проблемы ограниченного электроснабжения в труднодоступных местах и при экстремальных температурах. В отличие от классических решений на базе Li-ion аккумуляторов, данная система использует литий-ионный конденсатор, что обеспечивает устойчивость к морозам и колоссальный ресурс циклов зарядки.


##  Технические характеристики

| Параметр | Значение |
|---|---|
| **Микроконтроллер** | nRF52840 |
| **Операционная система** | Zephyr RTOS |
| **Источник питания** | Литий-ионный конденсатор (LIC) |
| **Интерфейсы** | BLE, GPIO, SPI/I2C |




##  Структура проекта
*   `boards/` — конфигурации для целевого оборудования.
*   `src/` — исходный код приложения и логика управления питанием.
*   `flash.sh` - скрипт для записи главного кода на микроконтроллер
