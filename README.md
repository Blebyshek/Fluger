# Микроконтроллерный сенсор с системой управления энергопотреблением

Проект автономного сенсорного узла, разработанного для работы в экстремальных условиях. Основная цель проекта — достижение максимальной автономности за счет использования литий-ионного конденсатора (LIC) и интеллектуальных алгоритмов управления питанием.
